# راهنمای نصب و راه‌اندازی سامانه مدیریت لید و لندینگ پوردانش

این پروژه یک سامانه کامل شامل لندینگ پیج هوشمند و پنل مدیریت پیشرفته برای مدیریت سرنخ‌ها (Leads)، پیگیری‌ها، امور مالی و محتوای سایت است.

## پیش‌نیازها
برای اجرای این پروژه نیاز به نصب موارد زیر دارید:
*   **Node.js** (نسخه ۲۰ یا بالاتر)
*   **npm** یا **yarn**

## مراحل نصب

### ۱. استخراج فایل‌ها
فایل‌های پروژه را در پوشه مورد نظر خود کپی کنید.

### ۲. نصب وابستگی‌ها
ترمینال را باز کرده و در مسیر اصلی پروژه دستور زیر را اجرا کنید:
```bash
npm install
```

### ۳. تنظیمات محیطی (اختیاری)
در صورت نیاز به استفاده از درگاه پیامک یا موارد خاص، می‌توانید فایل `.env` را ایجاد و تنظیم کنید. (نمونه در `.env.example` موجود است)

### ۴. اجرای پروژه در حالت توسعه
برای تست و مشاهده تغییرات به صورت زنده:
```bash
npm run dev
```
سپس آدرس `http://localhost:3000` را در مرورگر باز کنید.

### ۵. ساخت نسخه تولید (Production)
برای اجرای نهایی سایت روی سرور اصلی، ابتدا باید پروژه را بیلد کنید:
```bash
npm run build
```

### ۶. اجرای نسخه نهایی
پس از بیلد، با دستور زیر سامانه آماده به کار خواهد بود:
```bash
npm start
```

## اطلاعات ورود پیش‌فرض پنل مدیریت
*   **آدرس پنل:** `http://your-domain.com/admin`
*   **نام کاربری:** `admin`
*   **رمز عبور:** `admin123`

*(توصیه می‌شود بلافاصله پس از اولین ورود، رمز عبور را از بخش مدیریت ادمین‌ها تغییر دهید.)*

## ساختار دیتابیس
این پروژه از دیتابیس **SQLite** استفاده می‌کند که تمامی اطلاعات در فایل `database.sqlite` ذخیره می‌شود. نیازی به نصب سرور دیتابیس جداگانه (مانند MySQL) نیست.

## ویژگی‌های کلیدی
*   **لندینگ پیج:** دارای سیستم تایید شماره موبایل (OTP)، ویدیو پلیر هوشمند، تایمر معکوس و فرم ثبت‌نام داینامیک.
*   **مدیریت لیدها:** پروفایل پیشرفته برای هر مشتری، ثبت سوابق، پیگیری‌ها و مدیریت وضعیت.
*   **سیستم مالی:** ثبت واریزی‌ها (نقدی و اقساطی) به همراه آپلود فیش بانکی.
*   **شخصی‌سازی:** امکان تغییر تمامی متون، رنگ‌ها، فونت‌ها و بنرهای سایت از داخل پنل مدیریت بدون نیاز به کدنویسی.
