# راهنمای نصب و راه‌اندازی (آقای تحلیلگر)

این پروژه یک اپلیکیشن Full-Stack است که با React (فرانت‌ند) و Node.js/Express (بک‌ند) توسعه یافته و از SQLite برای ذخیره داده‌ها استفاده می‌کند.

## پیش‌نیازها
- نصب بودن **Node.js** (نسخه ۱۸ به بالا)
- نصب بودن **npm**

## مراحل نصب روی هاست (VPS یا هاست‌های اشتراکی با پشتیبانی Node.js)

### ۱. انتقال فایل‌ها
تمام فایل‌های پروژه را به هاست خود منتقل کنید (به جز پوشه `node_modules`).

### ۲. نصب وابستگی‌ها
در ترمینال هاست، دستور زیر را اجرا کنید:
```bash
npm install
```

### ۳. تنظیمات محیطی (Environment Variables)
فایل `.env.example` را به `.env` تغییر نام دهید و مقادیر دلخواه خود را وارد کنید:
- `ADMIN_PASSWORD`: رمز عبور برای ورود به پنل مدیریت (پیش‌فرض: `admin`)

### ۴. بیلد گرفتن (Build)
برای آماده‌سازی فایل‌های فرانت‌ند، دستور زیر را اجرا کنید:
```bash
npm run build
```
این دستور پوشه‌ای به نام `dist` ایجاد می‌کند.

### ۵. اجرای برنامه
برای اجرای برنامه در حالت عملیاتی، دستور زیر را بزنید:
```bash
npm start
```
برنامه به صورت پیش‌فرض روی پورت **3000** اجرا می‌شود.

## نکات مهم:
- **دیتابیس:** دیتابیس شما در فایلی به نام `database.sqlite` در ریشه پروژه ذخیره می‌شود. حتماً از این فایل بک‌آپ تهیه کنید.
- **خطای WebSocket:** اگر در محیط توسعه خطای WebSocket مشاهده کردید، نگران نباشید؛ این خطا مربوط به ابزار Vite است و در نسخه بیلد شده (Production) وجود نخواهد داشت.
- **مدیریت فرآیند:** پیشنهاد می‌شود برای اجرای دائمی برنامه روی سرور از **PM2** استفاده کنید:
  ```bash
  npm install -g pm2
  pm2 start server.ts --name "tahlilgar-app" --interpreter ./node_modules/.bin/tsx
  ```
  *(یا اگر از نسخه کامپایل شده استفاده می‌کنید، مستقیماً فایل سرور را با node اجرا کنید)*
